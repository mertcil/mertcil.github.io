# Product Context

## Why This Project Exists
- Present a credible, fast, and technically solid portfolio to support career opportunities.
- Provide a writing space (Posts) with safe Markdown rendering and consistent styling.

## Problems It Solves
- Consolidates experience, projects, and thoughts in one SEO-friendly site.
- Replaces ad-hoc styling with a consistent design system using MUI styled().

## Users & Goals
- Recruiters/Hiring Managers: Quickly assess experience, see selected projects, evaluate clarity and craft.
- Engineers/Peers: Skim technical choices, read posts, view code quality.

## Experience Principles
- Professional: Clear hierarchy, semantic headings, no visual noise.
- Fast: Server-first pages, minimal client JS, SSR styles to avoid flashes.
- Consistent: Shared primitives (Panel, Container, SectionTitle), tokens for color/typography.
- Accessible: Proper landmarks, color contrast, keyboard-friendly components.

## Content Strategy
- Home: Short hero with role and value.
- Work/Resume: Concise, scannable accomplishments and stack tags.
- Posts: Curated entries with validated frontmatter and readable typography.
